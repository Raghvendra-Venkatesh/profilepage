# Personal Profile Page
A simple personal profile webpage built using **HTML Frameset** and **CSS** for academic learning purposes.

## Features
- Header with profile photo and name  
- Navigation menu using frames  
- Separate pages for About, Objectives, Education, and Skills  
- External CSS for consistent styling
  
## Technologies Used
- HTML (Frameset)
- CSS
